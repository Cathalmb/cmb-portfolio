# cmb-portfolio

Cathal Mac an Bheatha — Product Designer & Creative Director  
Built with Eleventy + Nunjucks + Tailwind CSS

---

## Setup

```bash
npm install
```

## Development

Two terminals, run both:

**Terminal 1 — CSS:**
```bash
npm run dev:css
```

**Terminal 2 — Eleventy:**
```bash
npm run dev
```

Site runs at `http://localhost:8080`

## Build for production

```bash
npm run build:css
npm run build
```

Output goes to `/public` — this is what Vercel deploys.

## Deploy

```bash
git add .
git commit -m "your message"
git push origin main
```

Vercel picks it up automatically.

## Adding a new case study

1. Create `src/work/project-name.njk`
2. Copy the frontmatter from `bia-energy.njk` and update the values
3. Add images to `images/work/project-name/`
4. Add the card to the work grid in `src/index.njk`
5. `npm run build` and push

## Image guidelines

- Format: `.jpg` for photos, `.webp` for UI screenshots
- Hero images: 1600px wide max
- Grid images: 1200px wide max
- File size: under 300kb (use https://squoosh.app to compress)
- Naming: lowercase, hyphenated — `hero.jpg`, `brand-guidelines.jpg`
